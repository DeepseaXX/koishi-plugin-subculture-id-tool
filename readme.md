# koishi-plugin-subculture-id-tool

用于生成次文化特色网名/ID 的生成器。通过清晰的 **三层处理体系**，生成充满中二风、非主流色彩的独创 ID。

---

## 🏷️ 三层处理体系

插件在生成次文化 ID 时，会按照以下三个层级进行处理：

1. **核心变换层 (Core Transformation)**：对输入的关键词主体文字进行形变。
   * **[T1] 简转繁 (Traditional)**：将简体汉字转换为繁体汉字。
   * **[T2] 火星文 (Martian)**：将汉字转换为非主流火星文。
   * **[T3] 拼音化 (Pinyin)**：将中文字符转换为拼音字母。
   * **[T4] 极客文 (Leet)**：将英文字母/拼音转换为 Leetspeak 风格代码数字。
2. **文字伴侣层 (Text Embellishment)**：在变换文字的过程中，附加在其内部或首尾的修饰性符号。
   * **[D1] 火星包边 (Martian Border)**：火星文两端附带的修饰符对（如 `oοО ... Оοo`）。
   * **[D2] 拼音印记 (Pinyin Badge)**：拼音首尾的特殊符号（如 `★...★`）。
   * **[D3] 极客夹心 (Leet Spacer)**：极客文英文字符之间的连接符号（如 `ゞ`、`✿`）。
3. **首尾外挂层 (Outer Affixes)**：在已完成变换和点缀的主体词两端拼装的固定或随机词缀。
   * **[A1] 前置挂件 (Prefix)**：加在 ID 最前面的修饰词。
   * **[A2] 后置挂件 (Suffix)**：加在 ID 最后面的修饰词。

## 💬 指令列表

### 1. 生成 ID：`subid [keyword]`
生成一个专属的次文化特色 ID。如果未指定关键词，将提示输入。

#### 选项 (Options)
* `-p, --prefix <prefix>`：指定 **[A1] 前置挂件**。
  * 可输入数字编号选用预设（如 `-p 1`）。
  * 可输入 `none` 禁用前置挂件。
  * 可直接输入自定义前缀文本。
* `-s, --suffix <suffix>`：指定 **[A2] 后置挂件**。
  * 可输入数字编号选用预设（如 `-s 1`）。
  * 可输入 `none` 禁用后置挂件。
  * 可直接输入自定义后缀文本。
* `-t, --trans <trans>`：指定 **[T1-T4] 核心变换** 样式与叠加顺序。
  * 支持逗号分隔的标识符（如 `t1,t2`、`trad,mart` 或兼容数字编号 `1,2`）。
  * 输入 `none` 禁用核心变换，仅保留原始文本。
* `-d, --decor <decor>`：控制 **[D1-D3] 文字伴侣** 的点缀效果。
  * `none`：禁用所有文字伴侣点缀（去除火星包边、拼音印记、极客夹心，保留最纯粹的主体字变换效果）。
  * `mix`：开启火星包边左右随机混搭组合。
  * `pair`：开启火星包边左右对称配对。
  * `all` (或缺省)：使用插件全局配置。
* `-i, --iter <iter>`：指定核心变换的叠加迭代次数，硬上限为 10 次。
* `-l, --list`：显示支持的变换样式、伴侣点缀和首尾外挂列表及编号代号。

#### 示例
* `subid 雪芝麻糊 -p 1 -s none -t t3 -d none`
  固定首位预设前置挂件，禁用后置挂件，并应用拼音化核心变换，同时禁用文字伴侣点缀。
